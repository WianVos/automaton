~$ what is automaton

automaton is a lightweight external node classifier for puppet that supports multiple data backends. It has a simple command line interface, as well as an optional REST interface. automaton is written in ruby, and authored by shellfu, just another unix admin out there.

~$ features

  • command-line interface
  • rest interface
  • multi-node inheritance
  • parameterized classes
  • node fact storage via puppet's inventory service
  • facter fact interpolation
  • easy configuration via yaml
  • db or flat file storage
    • mongodb
    • yaml

~$ roadmap

  • configurable metadata storage (to be a bolt on/plugin)
  • documentation on writing a backend

~$ installation

lets go over the installation and configuration of automaton starting with an overview of steps. for more detailed steps please proceed to the github repository, and see the install.markdown file.

~$ requirements

  • ruby 1.8.7 or better
  • rails server if rest interface is used

~$ command-line interface

  1. clone the repository
  2. edit config.yml in your favorite editor

~$ rest interface

  1. create webserver config file
  2. create rails server config file
  3. restart/reload webserver and rails server

~$ optional tasks:

  1. create symlink for /usr/bin/automaton > /path/to/automaton/bin/automaton_cli.rb
  2. setup process monitoring
